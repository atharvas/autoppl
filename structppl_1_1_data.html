Detailed Description

template<class ValueType>
struct ppl::Data< ValueType >

Data is a light-weight structure that represents a set of samples from an observed random variable. It acts as an intermediate layer of communication between a model expression and the users. A Data object is different from a Param object in that it can hold multiple values but cannot be sampled. To this end, the user does not provide external storage for samples. It does not support set_value, but you can instead var.observe() to add an extra observation internally.
