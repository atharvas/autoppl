Detailed Description

template<class ValueType>
struct ppl::Param< ValueType >

Param is a light-weight structure that represents a univariate hidden random variable. That means the parameter does not hold samples, but it does contain a value that is used by model.pdf and get_value. Param requires user-provided external storage for samples and other algorithms. It is up to the user to ensure the storage pointer has enough capacity to support algorithms like metropolis-hastings which store data in this pointer. get_value supports an integer argument for compatibility with the get_value Data API, but this is never used.
